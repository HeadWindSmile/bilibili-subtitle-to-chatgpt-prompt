# Bilibili Subtitle to ChatGPT Prompt

一个用于 B 站视频学习的 Tampermonkey 油猴脚本。

本脚本可以在 B 站视频页面中自动提取当前视频的 CC / AI 字幕，并将字幕转换成适合 ChatGPT 总结的结构化 Prompt，一键复制到剪贴板。用户只需要打开 B 站视频，点击页面右下角按钮，即可快速生成用于 ChatGPT 的视频学习总结 Prompt。

## 功能特性

- 自动识别当前 B 站视频 BV 号
- 自动获取当前分 P 的 `cid`
- 自动读取 B 站 CC / AI 字幕列表
- 优先选择中文字幕
- 自动下载字幕 JSON
- 将字幕转换为带时间戳的文本格式
- 自动拼接高质量 ChatGPT 总结 Prompt
- 一键复制到剪贴板
- 页面右下角悬浮按钮操作
- Toast 提示当前执行状态
- 提供错误提示，方便排查无字幕、未登录、接口限制等问题

## 使用场景

这个脚本适合用于：

- 学习 B 站长视频课程
- 快速总结技术视频
- 提炼知识类视频重点
- 将视频内容转换为学习笔记
- 配合 ChatGPT 进行结构化总结
- 构建个人 AI 辅助学习工作流

## 安装方式

### 1. 安装 Tampermonkey

请先在浏览器中安装 Tampermonkey 插件。

推荐浏览器：

- Chrome
- Edge
- Firefox

### 2. 安装脚本

将仓库中的脚本文件复制到 Tampermonkey 中：

```text
bili-subtitle-to-chatgpt-prompt.user.js
```

或者在 Tampermonkey 中新建脚本，将文件内容粘贴进去并保存。

## 使用方法

1. 打开一个 B 站视频页面

```text
https://www.bilibili.com/video/BVxxxxxxx
```

2. 页面右下角会出现按钮：

   ```
   复制高密度总结 Prompt
   ```

   ![展示图片1](.\docs\展示图片1.png)

3. 点击按钮后，脚本会自动执行：

```text
获取视频信息
→ 获取当前分 P cid
→ 获取字幕列表
→ 下载字幕文件
→ 转换字幕文本
→ 构造 ChatGPT Prompt
→ 复制到剪贴板
```

4. 出现“已复制到剪贴板”提示后，打开 ChatGPT，将内容粘贴进去即可生成总结。

   ![展示图片2](.\docs\展示图片2.png)

## 输出 Prompt 包含内容

脚本生成的 Prompt 会要求 ChatGPT 输出：

- 极速速览区
- 一句话结论
- 重点速览
- 一屏总览卡
- 视频结构脑图
- 核心内容精讲
- 可执行行动项

目标是让用户在 ChatGPT 当前对话中，用较短时间看懂视频讲了什么、结构是什么、重点是什么，以及自己可以怎么用。

## 注意事项

### 1. 视频必须有 B 站 CC / AI 字幕

如果视频没有官方字幕、AI 字幕或 CC 字幕，脚本无法直接提取。

画面中由 UP 主压制进去的字幕不属于接口字幕，无法通过本脚本读取。

### 2. 部分视频可能需要登录

如果接口返回异常，可能是因为：

- 当前 B 站账号未登录
- 视频字幕接口需要登录态
- 当前视频无字幕
- B 站接口风控
- 字幕接口需要 WBI 签名

### 3. 长视频可能产生超长 Prompt

如果视频时间很长，字幕内容也会很长，可能超过 ChatGPT 单次输入长度限制。

后续可以增加自动分段总结功能。

## 常见问题

### Q1：为什么按钮出现了，但点击后提示没有字幕？

可能原因：

- 视频没有 CC / AI 字幕
- 字幕是 UP 主压在画面里的硬字幕
- 当前账号没有登录
- B 站接口没有返回字幕数据

### Q2：为什么复制后的内容很长？

因为脚本会把完整字幕和总结 Prompt 拼接在一起。长视频的字幕自然会比较长。

### Q3：可以直接总结没有字幕的视频吗？

当前版本不支持。
如果视频没有接口字幕，需要额外接入音频下载和 ASR 语音识别流程。

### Q4：支持多分 P 视频吗？

支持当前 URL 中的分 P。
脚本会根据 URL 参数中的 `p` 获取当前分 P 的 `cid`。

### Q5：支持自动发送到 ChatGPT 吗？

当前版本不支持。
当前版本只负责生成 Prompt 并复制到剪贴板，用户需要手动粘贴到 ChatGPT。

## 技术实现

核心流程：

```text
解析 BV 号
→ 请求 B 站视频信息接口
→ 获取当前分 P cid
→ 请求播放器字幕接口
→ 获取 subtitle_url
→ 下载字幕 JSON
→ 转换为带时间戳的字幕文本
→ 拼接结构化 Prompt
→ 复制到剪贴板
```

主要使用：

- JavaScript
- Tampermonkey
- GM_xmlhttpRequest
- GM_setClipboard
- Bilibili Web API
